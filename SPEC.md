# 回译法英语训练小程序 - 重构规格说明书

## 项目概述
- **项目名称**: 回译法英语训练
- **类型**: 多页面Web应用
- **核心功能**: 通过"原文→翻译→回译→Diff分析"的闭环训练，建立精准的双语转换能力
- **目标用户**: 英语学习者

## 页面架构

### 1. 首页 (home)
- Logo + 标题
- 核心配置卡片（MBTI、词汇量、置信度）
- 快速开始按钮
- 我的错题入口
- 题库管理入口
- 个人中心入口
- 设置入口（API Key等）

### 2. 学习页 (practice)
- 顶部进度条 + 返回按钮
- 阶段1: 英译中（原文→中文）
- 阶段2: 中译英（中文→英文回译）
- 反馈面板 + 完成面板

### 3. 题库页 (library)
- 题库列表展示
- 模式选择（日常/商务/学术/科技）
- 导入新语料（粘贴/文件）
- 难度筛选

### 4. 个人中心 (profile)
- 头像 + 用户信息
- 学习统计（总题数、准确率、连续天数、学习时长）
- 打卡日历
- 错题本入口
- 收藏列表

### 5. 错题页 (wrong-questions)
- 错题列表
- 错题详情
- 重新练习按钮

## 功能模块

### 配置管理
- MBTI类型选择（16种）
- 词汇量评估（4档+自定义）
- 学习偏好（可选）
- 置信度阈值（70-100%）

### 题库管理
- 内置20条语料（A2-C1难度）
- 用户导入语料
- 按主题/难度筛选
- 题库模式（日常/商务/学术/科技/文学）

### 学习流程
1. 英译中：原文→中文翻译
2. 中译英：中文→英文回译
3. 置信度判定（AI/规则）
4. Diff分析
5. 达标则进入下一题

### 判断模式
- AI智能判断（硅基流动/DeepSeek）
- 规则判断（本地）

### 统计系统
- 总学习题数
- 平均准确率
- 连续打卡天数
- 学习时长
- 薄弱点分析

## 验收标准
1. 首页正确显示所有入口
2. 配置可保存和加载
3. 学习流程完整（英译中→中译英）
4. AI判断正常工作
5. 统计数据正确显示
6. 题库可导入和管理
7. 错题可查看和复习
